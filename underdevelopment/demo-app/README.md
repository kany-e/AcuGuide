# AcuGuide Demo App

Static clickable demo web app for the hackathon presentation. This is a UI and flow prototype only: it does not implement live camera tracking, hand landmark detection, or medical logic.

## Run

Open `index.html` directly in a browser, or serve the folder with any static file server.

## Demo Path

Use the golden path:

Tension Headache -> Safety -> Routine Preview -> Camera Feedback -> Completion Recap

The coach screen uses a deterministic demo sequence and manual state buttons. It is intended to communicate the UI, product logic, and judge-facing flow without depending on real camera permissions or AI models.
