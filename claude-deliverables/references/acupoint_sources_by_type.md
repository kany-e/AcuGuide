# Hand & Wrist Acupoint Sources — Labeled by Type

**Type key:**
- **SPOT** — locates the point (where it is on the hand/wrist)
- **PROVE** — effectiveness study/review (does it work)
- **VALIDATION** — authority, credibility, or critical appraisal (how trustworthy the field is)
- **SAFETY** — contraindications / cautions

---

**One-line takeaway:** strongest **PROVE** is PC6 for nausea; **VALIDATION** sources agree location standards exist (WHO) but real-vs-sham efficacy is contested — so frame the product as guided self-care, not treatment.
